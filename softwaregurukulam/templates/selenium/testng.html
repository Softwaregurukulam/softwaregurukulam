Annotations

  • @BeforeSuite: This annotated method will be run before all tests in the suite.
  • @AfterSuite: This annotated method will be run after all tests in the suite.
  • @BeforeTest: This annotated method will be run before any test in the class.
  • @AfterTest: This annotated method will be run after any test in the class.
  • @BeforeGroups: This annotated method will run before the first test method in the group is invoked.
  • @AfterGroups: This annotated method will run after the last test method in the groups is invoked.
  • @BeforeClass: The annotated method will run before the first test method in the class is invoked.
  • @AfterClass: The annotated method will run after the last test method in the current class have been run.
  • @BeforeMethod: The annotated method will be run before each test method.
  • @AfterMethod: The annotated method will be run after each test method.
  • @Test: The annotated method is a part of a test case.

Execution sequence

  • BeforeSuite
  • BeforeTest
  • BeforeClass
  • BeforeMethod
  • Test
  • AfterMethod
  • AfterClass
  • AfterTest
  • AfterSuite

seleniumtest.java

package com;

import org.testng.annotations.Test;

public class seleniumtest
{
 @Test (groups = { "Car" })
 public void Car1()
 {
  System.out.println("Test car 1");
 }

 @Test (groups = { "Car" })
 public void Car2()
 {
  System.out.println("Test car 2");
 }

 @Test (groups = { "Bike" })
 public void Bike1()
 {
  System.out.println("Test bike 1");
 }

 @Test (groups = { "Bike" })
 public void Bike2()
 {
  System.out.println("Test bike 2");
 }

 @Test (groups = { "Car", "Bike" })
 public void bothcarandbike()
 {
  System.out.println("Test car & bike");
 }
}
