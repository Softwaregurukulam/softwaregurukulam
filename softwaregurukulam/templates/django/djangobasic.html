Django Basics

MVT Framework

URL

HTTP request use URL Mapper to send data to corresponding view

from django.conf.urls import url
urlpatterns = [
 url(r'^signup/', views.SignUp, name='signup'),
]

View

A view receives HTTP requests, access data via models and returns HTTP responses.

from django.http import HttpResponse
return render(request, 'signup.html')

Models

Models are Python objects defining an application's data structure. They provide options to create, edit and query data records in the database.

from django.db import models
class Category(models.Model):
 name = models.CharField(max_length=128, unique=True)
 designation = models.CharField(max_length=256, blank=True)
 def __str__(self):
  return self.name

Templates

Templates define the structure of a file layout to represent data in a web page.

Create project

django-admin startproject projectname

Create app

python manage.py startapp appname

Migration

python manage.py makemigrations
python manage.py migrate

Run server

python manage.py runserver

  • Using port: python manage.py runserver 8989
  • Using port: python manage.py runserver 127.10.10.10:8001

Configure the server IP using ALLOWED_HOST configuration in settings.py
ALLOWED_HOSTS = ['127.0.0.1','127.10.10.10:8001']

URL Pattern

from django.conf.urls import url
from . import views

urlpatterns = [
 url('app/2020/', views.year_2020), ## Each path is written with url()
 url('app/<int:year>/', views.year_record), ## Captured value is converted to int
 url('app/<int:year>/<int:month>/<slug:slug>/', views.year_detail), ## Captured value is converted to int & slug
]

Alternate Pattern Definition

  • (?P<name>pattern) named group
  • (pattern) unnamed group
from django.conf.urls import url
from . import views

urlpatterns = [
 url('app/(?P<year>[0-9]{4})/(?P<month>[0-9]{2})/(?P<slug>[\w-_]+)/', views.year_detail),
]

Grouping URL Pathern

  • /user/add
